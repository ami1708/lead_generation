Build plan — 4 phases

1

Foundation — Auth + Database persistence

Clerk login, Supabase database. Leads and scores saved permanently. Per-user data isolation.

2

Scoring engine — Python FastAPI pipeline

Parallel enrichment (Tavily) + scoring (Claude AI) for 1,000+ leads in under 30 seconds.

3

Automation — Weekly re-score + email alerts

Leads re-scored every week with fresh web data. Manager gets digest + instant alerts when scores jump.

4

Calling AI — Warm-up messages + call briefs (later)

Automated WhatsApp, personalised call briefs, auto CRM push after calls. Phase 2 of the product.
